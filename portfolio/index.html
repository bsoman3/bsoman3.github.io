Ninja Correlation of APT Binaries

I Gave a presentation at FIRST 2015, Berlin on evaluating the effectiveness of fuzzy hashing techniques in identifying the provenance of APT binaries. In this project, I considered three similarity metrics (Imphash, ssdeep, sdhash) for clustering APT binaries that share malware family, functionality or originating threat group. It used a data set of APT related binary samples from Jan-Mar 2015 and created clusters of similar binaries based on each of the three similarity metrics under consideration. It then evaluated the accuracy of the clusters and examined their implications on the effectiveness of each technique in identifying provenance of an APT related binary.

DNSSEC for Email Verification

Won the getdnsapi challenge at the TNW Hack Battle 2014 held in Amsterdam. With teammates Arvind and Ruslan, created a Thunderbird plugin that authenticates the sender domain of an email using DNSSEC and notifies the user of the verification result.

Gray-box analysis of web applications

Conceptualized a tool that uses static analysis of a web application to gather domain and type information about its input interfaces, and feeds this information to an automated crawler to achieve better test coverage of the application. Created a POC tool using C++ for generating control flow graphs, with a Java plugin for Burp Spider for fuzzing.
